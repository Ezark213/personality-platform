## 1. プロダクトコンセプト

### 中核UX：「決めつけない設計」

**課題認識**：
- 性格診断の良い点：「会話のきっかけ」「長所の発見」が上位
- 性格診断の悪い点：「性格を決めつけられるのが嫌だ」が約5割と突出

**本プラットフォームの方針**：
1. **診断結果を「ラベル」ではなく「傾向」として提示**
   - 「あなたはINTJです」→「今のあなたは内省的で戦略的な傾向が見られます」
   - 「状況で変わる」「成長可能」を常に併記

2. **行動設計に焦点**
   - タイプ名で終わらず、「今週の実験」（小さな行動提案）で終える
   - AI相談で毎回の会話を行動に落とす

3. **説明責任**
   - 診断の限界を認め、根拠を示す
   - 「何を根拠にどう出したか」を透明化

### 入口と出口の設計

```
入口（エンタメ・バイラル）         出口（意思決定支援）
┌─────────────┐         ┌─────────────┐
│ 恋愛・相性診断    │────────▶│ 恋愛相談（告白/謝罪） │
│ 無料・短時間     │         │ 会話プロンプト生成   │
│ SNS共有カード    │         │ 月980円〜        │
└─────────────┘         └─────────────┘
         │                          │
         │                          │
         ▼                          ▼
   週次サマリー ◀──────── 行動実験 ◀──── AI相談
     （無料）              （有料）      （回数制限）
```

### MBTI商標リスクの回避方針

**リスク**：
- 日本で「MBTI」は登録商標
- 「無料の公式MBTIは存在しない」と明言されている
- 流行している無料16タイプ診断は「MBTIではない」との注意喚起が継続

**対策**：
1. **「MBTI」を名乗らない**
   - サービス名・診断名に「MBTI」を使用しない
   - 独自ネーミング（例：「パーソナリティタイプ診断」「16の性格傾向」）

2. **学術的に広く使われる枠組みをベースにする**
   - Big Five（開放性、誠実性、外向性、協調性、神経症傾向）
   - 日本語尺度（TIPI-J等）が整備されている
   - 短縮尺度の研究蓄積もある

3. **注意書きの明記**
   - 「本診断は公式MBTIではありません」
   - 「心理学研究に基づく性格傾向の参考情報です」
   - 「重要な意思決定（就職・結婚等）には専門家への相談を推奨」

## 3. 診断テスト一覧（開発優先順）

### Phase 1: MVP（P0、2-4週間）

#### 1. 恋愛・相性診断 🔥

**優先理由**：バイラル性+課金率が最高

**機能**：
- 短縮版Big Five（20-30問、5分以内）
- 結果カード生成（1:1正方形 + 9:16ストーリーズ）
- 2人比較（招待リンク経由）
- 相性レポート：
  - 共通点/相違点
  - すれ違いが起きやすい場面
  - 会話プロンプト（言い換え例）

**バイラルループ**：
```
診断完了 → 結果カード → SNS共有 → 友人クリック → 診断開始
         ↓
    相性診断招待 → 友人が受検 → 相性レポート生成 → 再共有
```

**課金ポイント**：
- 無料：1人まで相性診断
- 有料：複数人の相性、詳細レポート（会話プロンプト集）

#### 2. 仕事・役割診断

**優先理由**：B2B転用の母体、課金率が高い

**機能**：
- 仕事上の強み・弱み
- 得意な役割（推進/調整/分析/支援）
- 上司/部下との相性
- 1on1台本生成

**B2B展開**：
- チーム相性可視化
- 配置シミュレーション
- マネジャー向け1on1ガイド

### Phase 2: 拡張（P1、5-8週間）

#### 3. コミュニケーション診断

**優先理由**：AI相談と直結、都度課金化しやすい

**機能**：
- 衝突パターン（怒り方/謝り方/頼み方）
- 文章テンプレ生成
  - 謝罪文
  - 依頼文
  - 断り方
  - 1on1台本

**AI相談連携**：
- テンプレ生成→AI相談で添削
- シーン別プロンプト（恋愛/仕事/家族）

#### 4. ストレス・回復診断

**優先理由**：継続価値（週次運用）に向く

**機能**：
- 燃え尽きパターン
- 回避癖
- 休息タイプ
- ストレストリガー

**継続施策**：
- 週次ストレスログ
- 回復行動の提案
- 月次振り返り

**注意**：
- 医療・診断を目的にしない旨を明記
- 深刻な状態の場合は専門家への相談を促す

### Phase 3: 最適化（P1-P2、9-12週間）

#### 5. 価値観診断

**優先理由**：マッチング/キャリアに展開可能

**機能**：
- 価値観の優先順位（お金/時間/承認/自由/安定/挑戦）
- キャリア方針（専門性/マネジメント/起業）
- 働き方の適性（リモート/出社/フレックス）

**展開先**：
- キャリアコーチング
- 転職サイト連携（アフィリエイト）
- B2B（組織文化マッチング）

## 8. バイラル設計

### 共有フォーマット（プロダクト要件として固定）

#### 1:1正方形カード（必須）

**仕様**：
- サイズ：1080×1080px
- 構成：
  - タイプ名（またはカテゴリ名）
  - 短いキャッチコピー（1行）
  - 強み3つ（箇条書き）
  - 「決めつけ禁止」短文（例：「これはあくまで傾向です。状況や成長で変わります」）
  - ロゴ・URL

**技術実装**：@vercel/og

#### 9:16ストーリーズ縦（必須）

**仕様**：
- サイズ：1080×1920px
- 構成：正方形カードの要素を縦長に再配置
- Instagram/TikTokストーリーズ用

#### X投稿テンプレ（必須）

**仕様**：
- 140〜200字程度
- 構成：
  - 診断結果（タイプ名）
  - あるある（共感ポイント）
  - 質問（「あなたは?」など）

## 9. リスク対策

### MBTI商標リスク

**リスク**：
- 日本で「MBTI」「Myers-Briggs」は登録商標
- 「無料の公式MBTIは存在しない」と明言されている
- 誤認を誘う表現は法的リスク

**対策**：
1. **「MBTI」を名乗らない**
   - サービス名・診断名・結果ページに「MBTI」を使用しない
   - 独自ネーミング（例：「16の性格傾向診断」）

2. **注意書き明記**
   - 「本診断は公式MBTIではありません」
   - 「心理学研究に基づく性格傾向の参考情報です」

3. **学術枠組みの明示**
   - Big Fiveベースであることを明記
   - 参考文献・尺度の出典を記載

### 決めつけ嫌悪

**リスク**：
- 「性格を決めつけられるのが嫌」が約5割
- ラベリングによる自己固定化、他者誤解

**対策**：
1. **結果表現の統一**
   - 「あなたは〇〇です」→「今のあなたは〇〇の傾向が見られます」
   - 「傾向」「状況で変わる」「成長可能」を常に併記

2. **行動設計に焦点**
   - タイプ名で終わらず、「今週の実験」で終える
   - AI相談で行動改善に繋げる

3. **UI上の工夫**
   - 結果ページに「この結果は固定ではありません」を明示
   - 再診断を推奨（3か月ごと）

### 課金トラブル

**リスク**：
- 返金/保証運用への不満
- サブスク解約導線が不明確
- 自動更新への認識不足

**対策**：
1. **返金ポリシー明記**
   - 7日間返金保証（初回のみ）
   - 返金条件を明示（未使用のクレジット等）

2. **解約導線の明確化**
   - ダッシュボードから1クリックで解約ページへ
   - 解約理由のフィードバック収集（任意）

3. **トライアル期間の適切な設計**
   - Day0に価値を体験させる
   - トライアル終了前に通知（3日前、1日前）

4. **Stripe Billing Portal活用**
   - ユーザー自身で支払い方法変更、請求履歴確認、解約が可能

### プロファイリング・透明性

**リスク**：
- 性格特性はセンシティブ情報
- 自動化された意思決定への懸念（GDPR等）

**対策**：
1. **透明性**
   - 「何を根拠にどう出したか」を説明
   - スコア計算式の概要を公開
   - AI相談のプロンプト設計の方針を明示

2. **ユーザーコントロール**
   - データの編集権
   - 削除権（アカウント削除時に全データ削除）
   - 学習利用のオプトアウト

3. **利用用途の制限**
   - 「娯楽/自己理解/自己改善」に限定
   - B2Bでも「採用合否の自動決定」には使わない旨を契約に明記

4. **プライバシーポリシー**
   - データ保存期間の明示
   - 第三者提供の有無（原則なし）
   - 広告利用の有無（原則なし）
