# 収益化戦略詳細

## 💰 価格設定の根拠

### 競合分析
- **16Personalities** - 無料 + プレミアム $29
- **トゥルーカラーズ** - $50〜
- **CliftonStrengths** - $49.99
- **エニアグラム研究所** - $12〜

### 価格設定の理由
**¥980/月（プレミアム）**
- Netflix（¥1,490）より安い
- 自己投資として受け入れやすい価格帯
- AI相談機能の価値を考慮
- 年払いで20%割引（¥9,800）でLTV向上

## 📊 ユニットエコノミクス

### ユーザー獲得コスト（CAC）
- 初期: ¥2,000〜3,000（広告・マーケティング）
- オーガニック流入増加後: ¥500〜1,000

### 顧客生涯価値（LTV）
- プレミアムユーザー平均継続: 6ヶ月
- LTV = ¥980 × 6 = ¥5,880
- LTV/CAC比率 = 5,880/2,000 = 2.94（健全）

### 損益分岐点
- 固定費（月）: ¥150,000（サーバー、AI API、開発）
- 必要プレミアムユーザー: 154人
- 必要総ユーザー（転換率2%）: 7,700人

## 🎯 KPI（重要指標）

### 北極星指標
**月間アクティブ診断数**（診断を完了したユーザー数）

### その他重要KPI
1. **獲得**
   - 月間新規登録数
   - 流入チャネル別ユーザー数
   - CAC

2. **アクティベーション**
   - 初回診断完了率
   - 登録後24時間以内の診断率

3. **リテンション**
   - DAU/MAU比率
   - 診断リピート率
   - 7日後/30日後定着率

4. **収益**
   - 無料→プレミアム転換率
   - ARPU（ユーザーあたり平均収益）
   - MRR（月間経常収益）
   - チャーン率

5. **紹介**
   - バイラル係数
   - 診断結果シェア率

## 🚀 収益化タイムライン

### Phase 1（1-2ヶ月）: トラクション獲得
- **目標**: プロダクトマーケットフィット検証
- **施策**: 全機能無料提供
- **KPI**: 1,000人の診断完了
- **収益**: ¥0（投資期間）

### Phase 2（3ヶ月目）: ソフトローンチ
- **目標**: 収益化テスト
- **施策**:
  - プレミアム機能リリース
  - 既存ユーザーに特別価格（¥500/月）
  - AI相談を課金機能として試験導入
- **KPI**: 転換率1%以上
- **収益目標**: ¥50,000/月

### Phase 3（4-6ヶ月）: 本格収益化
- **目標**: 持続可能な成長
- **施策**:
  - 正規価格（¥980/月）
  - 単品課金オプション追加
  - リファラルプログラム開始
- **KPI**: 転換率2-3%
- **収益目標**: ¥200,000-500,000/月

### Phase 4（7-12ヶ月）: スケール
- **目標**: B2B展開
- **施策**:
  - 企業向けパッケージ
  - API提供
  - アフィリエイトプログラム
- **収益目標**: ¥1,000,000+/月

## 🎁 グロースハック施策

### 1. バイラルループ
```
診断完了 → 結果がすごく当たってる！
→ SNSでシェア → 友人が見る
→ 「自分もやってみたい」→ 新規登録
```

**実装**
- 結果ページにシェアボタン（Twitter、LINE、Instagram）
- OGP画像を動的生成（診断結果を含む）
- 「友達と比較」機能

### 2. リファラルプログラム
- 紹介した人: 1ヶ月無料
- 紹介された人: 初月50%オフ
- 紹介リンクを簡単に生成

### 3. コンテンツマーケティング
- 「MBTI別おすすめキャリア」などのブログ記事
- SEO最適化
- 診断へのCTAを自然に配置

### 4. メールマーケティング
**オンボーディングシーケンス**
1. 登録直後: ウェルカムメール
2. +1日: 初回診断の促し
3. +3日: 他の診断の紹介
4. +7日: AI相談機能の案内
5. +14日: プレミアムへのアップグレード提案

### 5. ゲーミフィケーション
- バッジ・実績システム
- 診断コンプリート率
- レベルアップ制度

### 6. リエンゲージメント
- 診断していないユーザーへのリマインダー
- 新しい診断リリース通知
- 「あなたと似た人はこんな診断をしています」

## 🚨 リスクと対策

### リスク1: ユーザー獲得コストが高い
**対策**: オーガニック流入（SEO、バイラル）に注力

### リスク2: 転換率が低い
**対策**: 段階的な価格設定、トライアル期間、単品課金オプション

### リスク3: AI APIコストが高騰
**対策**: キャッシング戦略、トークン最適化、プレミアム機能限定

### リスク4: 競合出現
**対策**: AI相談機能による差別化、コミュニティ形成、継続的な診断追加
