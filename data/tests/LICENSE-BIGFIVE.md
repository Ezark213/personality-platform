## Our Use

We have adapted the BigFive OSS Japanese question data (120 items) for use in our personality assessment platform. The data has been:

1. Downloaded from the BigFive Web repository
2. Converted to our internal data format
3. Used to create both full (120-item) and short (20-item) versions of the Big Five personality test

We gratefully acknowledge the BigFive project and IPIP for making this high-quality, scientifically validated question data available to the community.
