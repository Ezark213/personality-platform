# Personality Platform - プロジェクト計画書

## 🎯 プロジェクト概要

複数の性格診断テストを提供し、診断結果をAIに相談できる総合プラットフォーム

## 📋 提供する診断テスト

### Phase 1（MVP）
1. **ラブタイプ診断** - 恋愛傾向の分析
2. **MBTI診断** - 16タイプ性格診断
3. **ビッグファイブ診断** - 開放性、誠実性、外向性、協調性、神経症傾向

### Phase 2（拡張）
4. **適性検査** - キャリア・職業適性
5. **ストレングスファインダー風** - 強み診断
6. **エニアグラム** - 9つの性格タイプ

## 🏗️ 技術スタック

### フロントエンド
- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **React Hook Form** + **Zod** (フォームバリデーション)

### バックエンド
- **Next.js API Routes** / **Server Actions**
- **Prisma** (ORM)
- **PostgreSQL** (Supabase推奨)

### AI統合
- **OpenAI GPT-4** または **Anthropic Claude**
- **Vercel AI SDK** (ストリーミング対応)

### 認証
- **Clerk** または **NextAuth.js v5**

### デプロイ
- **Vercel** (Next.js最適化)
- **Supabase** (DB + Storage)
